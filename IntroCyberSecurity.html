信息安全（网络空间安全）导论

期望能够通过尽量短小的介绍，使同学能够快速了解计算机极简发展过程，计算机运行的直观概念性原理，信息安全的研究及其大致的内容等等，使同学们对信息安全或网络空间安全有一个大致的了解，如雾里看山，隐隐绰绰。

作业一

注：工欲善其事,必先利其器，推荐大文件下载利器XTREME DOWNLOAD MANAGER，网址：https://xtremedownloadmanager.com/

  1. 下载安装开源虚拟机软件Virtual Box。网址https://www.virtualbox.org/
  2. 在Virtual Box中安装Linux的Ubuntu发行版。Ubuntu的下载地址https://cn.ubuntu.com/download/desktop ，从Ubuntu的官网上下载比较慢，也可以找国内的镜像网站，如网易开源镜像站（http://mirrors.163.com/）下载，或者从我的校园网盘上的共享下载（分享链接: 链接:http://pan.buu.edu.cn/share?id=5xq8vxg3myru）。
    注：校园网盘上的共享通常不是最新的，最新的软件需要到官方网站上下载。
  3. 为了以后利用apt安装软件能快一些，建议可以把国内的几个源（如阿里、腾讯、清华、中科大等）加到apt的配置文件“/etc/apt/sources.list”中。
  4. 在你安装的Ubuntu操作系统上，在终端内执行下面的指令安装nmap软件。
    sudo apt install nmap
    安装结束后，在终端执行以下指令，如果能够显示nmap的指令说明，则表示安装成功。
    nmap -h
  5. 在Ubuntu操作系统的软件中心“Ubuntu Software”中查找并安装NmapSI4（这是Nmap的一个图形化界面）。
  6. 在终端内执行下面的指令安装net-tools软件。
    sudo apt install net-tools
    安装结束后，在终端执行以下指令，查看本机ip地址。(B站视频演示)
    ifconfig
  7. 启动nmapsi4软件，选择"Quick Scan“模式，在软件右侧的参数输入部分左上的第一个框内输入你的ip地址，然后按界面左上的“scan”键进行扫描。(B站视频演示)
  8. 在你的宿主机操作系统中（就是安装Virtual Box的系统）安装一个录屏软件。
  9. 使用录屏软件录制以下内容：
    • Ubuntu在虚拟机VirtualBox中启动的过程；
    • 启动后执行终端程序（也就是进入命令行），执行ifconfig指令。
    • 启动nmapsi4，并执行上述扫描的过程。
  10. 录制的视频软件存为mp4格式，文件名为形式为：****-###.mp4，其中****为你的学号，###为你的中文姓名。
  11. 可在联大云盘(https://pan.buu.edu.cn/)中搜索群组“李晓峰的课程作业”，加入群组，上传你的文件。
    如果学生在校外，可按学校由外网连接内网的方式，先连到内网，再访问联大云盘。
  12. 作业截止时间在上课时告知。
