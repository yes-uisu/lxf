计算机语言发展史（整理自：https://en.wikipedia.org/wiki/History_of_programming_languages）

第一阶段：First programming languages
1951 C Regional Assembly Language
1952 C Autocode
1954 C IPL (forerunner to LISP)
1955 C FLOW-MATIC (led to COBOL)
1957 C FORTRAN (First compiler)
1957 C COMTRAN (precursor to COBOL)
1958 C LISP
1958 C ALGOL 58
1959 C FACT (forerunner to COBOL)
1959 C COBOL
1959 C RPG
1962 C APL
1962 C Simula
1962 C SNOBOL
1963 C CPL (forerunner to C)
1964 C Speakeasy (computational environment)
1964 C BASIC
1964 C PL/I
1966 C JOSS
1967 C BCPL (forerunner to C)


第二阶段：Establishing fundamental paradigms
1968 C Logo
1969 C B (forerunner to C)
1970 C Pascal
1970 C Forth
1972 C C
1972 C Smalltalk
1972 C Prolog
1973 C ML
1975 C Scheme
1978 C SQL (a query language, later extended)

第三阶段：1980s: consolidation, modules, performance
1980 C C++ (as C with classes, renamed in 1983)
1983 C Ada
1984 C Common Lisp
1984 C MATLAB
1985 C Eiffel
1986 C Objective-C
1986 C Erlang
1987 C Perl
1988 C Tcl
1988 C Wolfram Language (as part of Mathematica, only got a separate name in June 2013)
1989 C FL (Backus)

第四阶段：1990s: the Internet age
1990 C Haskell
1991 C Python
1991 C Visual Basic
1993 C Ruby
1993 C Lua
1993 C R
1994 C CLOS (part of ANSI Common Lisp)
1995 C Ada 95
1995 C Java
1995 C Delphi (Object Pascal)
1995 C JavaScript
1995 C PHP
1997 C Rebol
1999 C D
