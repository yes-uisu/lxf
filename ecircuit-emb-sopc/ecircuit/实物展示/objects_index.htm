说明

  CAM350软件（Downstream公司）

　CAM350 is a tool that bridges the gap between CAD and CAM, and is the first CAM product built around an intelligent CAD database.

示例：CAM350 version 10.7 build 560

打开示例CAM文件

在一些图像对象的表述上，已经逐渐接近生产内容，比如以前的过孔，现在表述就是drill（钻）。

CAM350/DFMStream version 12.1 Build 1017：CAM350就是把layout工程师设计出来的线路板，经客户以电脑资料的方式给线路板厂，然后板厂根据该厂里的机器设备能力和生产能力，利用CAM软件（genesis2000,cam350,ucam,v2001等）将客户提供的原始资料根据该厂的生产能力修正后，为生产的各工序提供某些生产工具（比如菲林、钻带、锣带等），以方便本厂能生产符合客户要求 的线路板，起的就是辅助制造作用。

BluePrint（Downstream公司）

（官网说明）BluePrint-PCB is DownStream’s PCB Documentation Solution.

BluePrint-PCB version 5.1 Build 1117：BluePrint 是DownStream公司的针对PCB生产和装配流程的指导和管理软件。该软件提高编写PCB设计文档的效率，让PCB设计工程师花更多的时间在他们需要提高的PCB设计能力上，而花更少的时间去编写低价值但是又很重要的PCB设计文档上。通过记录工程设计意图要求工程按指定的形式、安装和完成制成品，指导採购进程计划，控制对制成品的检验从而保证将来产品生产的质量一致性。BluePrint缩短了我们创建PCB设计文档的时间，并且减小了生产的风险。
