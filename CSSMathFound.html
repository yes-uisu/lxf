《网络空间安全数学基础》

此网页中的《网络空间安全数学基础》为一学期的本科课程。

自编教材

  • 发布版PDF（FoCysMath）
  • Latex源码

教材包括以下内容的基础知识。

  1. 预备知识：集合、关系、函数、势、皮亚诺算数公理系统、拓扑空间、组合数学。
  2. 整除：最大公因子、最小公倍数、算数基本定理等。
  3. 同余：剩余系、乘法逆元、费马小定理、威尔逊定理、线性同余方程、同于方程组求解。
  4. 二次剩余：二次同余方程、欧拉判别条件、勒让德符号、雅克比符号。
  5. 原根与指数：次数、原根、指数。
  6. 群：子群、阿贝尔群、循环群、商群、同态、同构。
  7. 环：子环、同态、同构、商环。
  8. 域：子环、同态、同构。

编程练习

以下顺序，也是实际授课时编程练习的顺序。除了第一埃拉托色尼算法练习，剩下的练习是一个系列练习，先实现一个交互式命令行框架，然后在此框架下，依次实现不同的算法。
  1. EratosthenesAlg：编程实现埃拉托色尼 (Eratosthenes，古希腊数学家)筛选法寻找素数，按一定的步长计算一系列不同范围内的素数(500∼1000000)，并记录算法所用时间，并画出不同范围时算法所用时间的变化曲线。注：不同范围搜寻的时间可以以一定的格式写到文本文件中，比如 34,500 < 换行 >，每一个搜寻范围形成这样一行数据，然后将这个数据文件导入 excel，用 excel 生成图形。 任务说明(reame)， 作业初始仓库， )
  2. 最大公约数gcdalg:利用C 语言已有的运算，利用欧几里得算法实现求解 n 和 m 的最大公约数。在编写时，要意识到计算位数的限制。任务说明(readme)， 作业初始仓库， )
  3. 框架:实现一个交互式命令行程序的框架。 任务文档(PDF)， 作业初始仓库 )
  4. prime_erat(n,m):利用 Eratosthenese筛选法，从屏幕输出从 n 到 m 的所有素数，如果此范围内不存在素数，输出none。 任务文档(ex.PDF)， 作业初始仓库 )
  5. gcd(n,m):利用GMP大数库，实现两个数最大公约数的计算。 任务文档(ex.PDF)， 作业初始仓库 )
  6. lcm(n,m):利用GMP大数库，求解 n 和 m 的最小公倍数的算法。 任务文档(ex.PDF)， 作业初始仓库 )
  7. factor(n):求解整数n的标准分解式。 任务文档(ex.PDF)， 作业初始仓库 )
  8. eulerfun(n):计算 n 的欧拉函数。 任务文档(ex.PDF)， 作业初始仓库 )
  9. inverse(n,m):实现扩展欧几里得算法，求解 n 模 m 的逆元，如果不存在，输出 none。。 任务文档(ex.PDF)， 作业初始仓库 )
  10. crt(a,b,c,d,e,f) :实现CRT算法。 任务文档(ex.PDF)， 作业初始仓库 )
  11. order(a,m) :a 对模 m 的次数。 任务文档(ex.PDF)， 作业初始仓库 )
  12. primroot(m) :求 m 的原根。 任务文档(ex.PDF)， 作业初始仓库 )
  13. legendresym(a,p) :利用二次互反律，计算a对p的legendre符号。 任务文档(ex.PDF)， 作业初始仓库 )

编程练习评价

  • 发布版PDF

对于编程类练习，应该有个评价指导过程，此课的评价方法所对应的条件是一个授课老师的情况，本着评分尽量公平合理的原则下制定的，对于一些练习，学生练习结果不理想，应采取指导评价再指导再评价的反复循环，一个练习多个循环。

对于有条件的学校，建议配几个助教，评分方法可以改变，可以形成对学生有针对性的个性化指导、提高。
